# golang面试前的准备

## 前序

```text
一个底层程序员的悲哀

我现在的公司算是很好吧。弹性、不打卡、请假不扣钱、没有考勤····这么看怎么都是神仙公司

如果不是因为一些事。谁会傻到想不开。

说下我的领导吧。对我特别好！平时喜欢鼓励我，为了我的成长让我学更多的东西。

因为公司没有具体的规章制度。领导口头的话大家都默认遵守。

省略一部分······

客观评价下。公司做的产品不咋滴。技术落后了好几年···一个框架用了5年都没更新！

一次跟大厂朋友聊天的过程中得知了一些比较好的技术。可以改进我们现有系统的技术。

我花了两周自己的业余时间搭建了一套demo给领导看。我对他说这个框架跟我们现有系统结合起来可以节省最少30%的人力，大大的提高效率并且做了相关演示。

领导很开心。我说这套框架是付费的。源码买断10000￥。领导说你写完了我给你报销(口头+文字记录)

于是我自费 "私自采购" ，花了自己两个月时间(下班时间+周六日)终于完成了80%。

我再次去邀功。领导说大老板知道了这个。后期给我报销········

为了避免后期移植浪费时间。我自费买了(期间请朋友帮我demo。技术咨询花费3000)

中途被调去别的项目组。我领导仿佛忘了报销这个事情。

直到今年我直接告诉他有份报销没给我。但是他。不认账了！对。就是不认账了。还反咬一口说我不遵守规章制度私自采购！

说句自夸的话：现在公司的系统4个开发写了4个月现在还没写完。全是bug！我用这套新的方案花了一周前后端独立重构完成30%

一周以后我给演示效果，证明的我的方案更快。提升的效率不止30%。我以为他会同意给我报销并使用这套新的架构。但是没有。给我画饼2小时。

忍无可忍。决定正式摆烂。每天都为下一份工作做准备。

口口声声说规章制度。我现在都不知道规章制度是什么。貌似没有同事知道。。


1. 还有就是承诺我的涨薪。两年未到账。我一直信了他的话。公司没赚钱(不经意间得知领导雅迪换了雅阁)

2. 承诺用我的云服务器补贴未到账！今天告诉我说咱们有你还用自己的本身就不符合规章制度

3. 公司电脑配置低不给升级。用自己的Mac两年。坏了自己修。没有一分钱补贴！

4. 去年经常凌晨打电话。打车去公司还不报销

5. 周六日加班自己花钱买饭

产品没创新！代码没规范！天天画大饼！

今天给我画完饼还不停的pua我。还想白嫖我的方案。还想让我给他写详细文档。他好去“说服”其它领导！怎么会有这么厚颜无耻之人。

我还皮笑肉不笑的给算了笔账。10个程序员。每个人提高30%的工作效率。。10000块钱分期24个月来算。一个月只需要500块。

不经意看到公司去年赚钱了。领导薪资好像都翻倍了

```
